====================================================
  CA ARTICLE TOOLKIT — HOW TO USE
====================================================

FIRST TIME SETUP (only once per computer):
-------------------------------------------
1. Make sure Python is installed.
   → Download from: https://www.python.org/downloads/
   → During install, TICK ✅ "Add Python to PATH"

2. Double-click: "INSTALL PACKAGES (First Time Only).bat"
   → This installs Streamlit and other required libraries
   → Wait for it to finish, then close the window

RUN THE APP (every time):
--------------------------
1. Double-click: "Start CA Toolkit.bat"
   → The launcher window will appear
   
2. Click the "Start Toolkit" button
   → Your browser will open automatically
   → Login at: http://localhost:8501

LOGIN CREDENTIALS:
------------------

  (Admin can change passwords inside the app under Admin Panel)

STOPPING THE APP:
-----------------
  Click "Stop Toolkit" in the launcher window,
  then close the window.

FOLDER CONTENTS:
-----------------
  app.py                          → Main Streamlit application
  launcher.py                     → GUI launcher
  users.json                      → Login credentials (auto-updated)
  Start CA Toolkit.bat            → Double-click to start
  INSTALL PACKAGES (First Time).bat → Run once for setup
  README.txt                      → This file
